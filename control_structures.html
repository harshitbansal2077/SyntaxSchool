Introduction

This tutorial covers control structures, including if statements and loops.

If Statements

You can use if statements to execute code based on conditions:

if age >= 18:
print("You are an adult.")

Loops

Loops allow you to execute a block of code multiple times. Here's an example of a for loop:

for i in range(5):
print(i)
