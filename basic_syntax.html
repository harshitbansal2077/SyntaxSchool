Introduction

In this tutorial, you will learn about Python's syntax and how to work with variables.

Variables

Variables are used to store data. Here’s how you can create variables in Python:

name = "John Doe"
age = 30
is_student = True

Printing Variables

To print the variables, use the print() function:

print(name)
print(age)
print(is_student)
