Introduction

This tutorial introduces you to modules and packages in Python.

Importing a Module

You can import a module using the import statement:

import math
print(math.sqrt(16)) # Output: 4.0

Creating a Module

You can create your own module by saving your functions in a .py file:

# my_module.py
def add(a, b):
return a + b
